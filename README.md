# DevStudio — Landing Page Profissional

Projeto de uma landing page moderna para uma agência digital fictícia, desenvolvido para praticar HTML5 semântico, CSS3 responsivo e interações com JavaScript.

## Tecnologias

- HTML5
- CSS3
- JavaScript

## Funcionalidades

- Layout moderno e responsivo
- Seções organizadas (Hero, Serviços, Sobre, Contato)
- Formulário com validação em JavaScript
- Boas práticas de estrutura e semântica

## Preview

Interface profissional simulando site real de empresa.
